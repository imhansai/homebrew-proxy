# Imhansai Tool

## How do I install these formulae?

`brew install imhansai/tool/<formula>`

Or `brew tap imhansai/tool` and then `brew install <formula>`.

## Documentation

`brew help`, `man brew` or check [Homebrew's documentation](https://docs.brew.sh).
