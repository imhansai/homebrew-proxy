# Imhansai Proxy

## How do I install these formulae?

`brew install imhansai/proxy/<formula>`

Or `brew tap imhansai/proxy` and then `brew install <formula>`.

## Documentation

`brew help`, `man brew` or check [Homebrew's documentation](https://docs.brew.sh).
